✅ La página Viaje Capurrense funciona como una invitación digital para reunir a un grupo de ami
✅ Funcionalidad principal:
Presenta un mensaje inspirador sobre el reencuentro y la amistad.
Utiliza una narrativa enfocada en transformar los años y la distancia en una oportunidad para volver a reunirse.
Refuerza la identidad del grupo mediante referencias al barrio de Capurro.
Incluye una llamada a la acción asociada a WhatsApp para facilitar el contacto o la coordinación del viaje.
